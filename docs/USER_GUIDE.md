# GochiSpace ユーザーガイド

**美味しい発見をシェアする、信頼できる仲間だけの飲食店レビューサービス**

> **注意：** このガイドはサービスの現時点の仕様をもとに作成しています。サービスの基本的な方向性は変わりませんが、機能の細部（画面の構成・項目名・操作手順など）は今後のアップデートで変更される場合があります。最新の情報はアプリ内の表示をご確認ください。

---

## GochiSpace とは？

GochiSpace は、家族・友人など**信頼できる人同士だけ**で飲食店レビューを共有できるサービスです。

匿名のレビューサイトと違い、**あなたが知っている人の口コミ**だけが届きます。相手の好みや行動パターンを知っているからこそ、「あの人がおすすめするなら間違いない」という安心感でお店を選べます。

---

## はじめる前に

- スマートフォン・PCのブラウザから利用できます
- アカウント登録にはメールアドレスが必要です
- 友人・家族を招待してフォローし合うことで、レビューを閲覧できるようになります

---

## 利用の流れ（全体像）

```
1. アカウント登録・ログイン
      ↓
2. ユーザidとユーザー名を設定（後から変更可能）
      ↓
3. 友人・家族などの知り合いと相互フォローし合う
      ↓
4. 飲食店のレビューや行きたい店を投稿・閲覧する
      ↓
5. 地図でお店を探す
```

## 各機能の使い方

### 1. アカウント登録・ログイン

1. トップページにアクセスします
2. **「新規登録」** でメールアドレスとパスワードを入力してアカウントを作成します
3. すでにアカウントをお持ちの場合は **「ログイン」** から入ります

---

### 2. プロフィール設定（初回）

初めてログインすると、プロフィール設定画面が表示されます。

| 項目 | 説明 |
|------|------|
| ユーザー名 | 英数字・アンダースコアで設定（例: `taro_yamada`）。他のユーザーがあなたを検索するときに使います |
| 表示名 | アプリ上に表示される名前（例: 山田太郎） |

入力後、**「はじめる」** をタップするとサービスを使い始められます。

> プロフィールはあとからいつでも変更できます（プロフィール画面から）。

---

### 3. 好きなジャンルを登録する（任意）

プロフィール画面から、好きな料理ジャンルを**最大3つ**選べます。

対応ジャンル（変更になる場合があります）：

- ラーメン / イタリアン / 中華 / 焼肉 / 寿司
- 和食 / カフェ / バー / フレンチ / タイ料理
- 韓国料理 / インド料理 / ピザ / バーガー / 居酒屋
- ファストフード / シーフード / その他

登録しておくと、将来的なレコメンド機能などに活用されます。

---

### 4. 友人・家族をフォローする

GochiSpace では**相互フォロー**が成立して初めてレビューを共有できます。

#### フォロー申請の手順

1. 下部メニューの **「フォロー」** をタップします
2. **「ユーザーを検索」** 欄に相手のユーザー名を入力して検索します
3. 検索結果に表示されたユーザーの **「フォロー申請」** をタップします
4. 相手が申請を承認すると、相互フォローが成立します

#### フォロー申請の承認・拒否

1. 下部メニューの **「フォロー」** をタップします
2. **「受け取った申請」** セクションに届いた申請が表示されます
3. **「承認」** または **「拒否」** を選びます

#### 関係性（ロール）の設定

フォロワーに対して関係性を設定できます。

| ロール | 例 |
|--------|-----|
| 家族 | 配偶者・親・兄弟 |
| 友達 | 仲の良い友人 |
| 知人 | 顔見知り程度 |
| 上司 | 職場の上司 |
| 先輩 / 後輩 | 職場・学校での先輩後輩 |
| パートナー | 恋人・パートナー |

フォロー一覧から各フォロワーのロールを変更できます。

---

### 5. レビューを投稿する

#### 飲食店を記録する（レビュー投稿）

1. 下部の **「飲食店を記録」** ボタンをタップします
2. **STEP 1：お店を検索する**
   - お店の名前を入力して検索します
   - 候補一覧からお店をタップして選択します
3. **STEP 2：評価を入力する**

   | 項目 | 説明 |
   |------|------|
   | 評価 | 「また行きたい」「普通」「好みじゃなかった」の3段階 |
   | ジャンル | お店のジャンル（ラーメン・カフェなど） |
   | シチュエーション | 誰と行ったか（デート・家族の食事・一人飯など） |
   | コメント | 自由記述のメモ |
   | 訪問日 | いつ行ったか |
   | 写真 | 最大3枚までアップロード可能 |

4. **「投稿する」** をタップして完了です

> 投稿したレビューは、**相互フォロワー全員**に共有されます。

---

### 6. フィードでレビューを閲覧する

下部メニューの **「フィード」** から、フォローしている人たちのレビューを確認できます。

#### フィードの見方

- **「レビュー」タブ** ：フォロワー全員のレビューをお店ごとにまとめて表示（ロール・シチュエーション・ジャンルで絞り込み可能）
- **「行きたい」タブ** ：自分が「行きたい」と登録したお店の一覧

各レビューカードには以下の情報が表示されます：

- お店の名前・ジャンル
- 評価（また行きたい / 普通 / 好みじゃなかった）
- コメント・訪問日・シチュエーション
- 投稿した人の表示名
- 写真（投稿されている場合）

#### レビューカードでできること

- カードをタップすると、そのお店に投稿されたレビューの詳細が表示されます
- **「誘う」ボタン** ：フォロワーに「一緒に行きたい」リクエストを送れます
- 詳細表示の **「ルート」ボタン** ：Google マップでお店へのルートを開きます
- 自分のレビューは詳細表示のゴミ箱アイコンから **削除** できます

---

### 7. 地図でお店を探す

下部メニューの **「マップ」** から、地図上でフォロワーがレビューしたお店を探せます。

- 地図上のマーカーをタップすると、そのお店のレビューが表示されます
- 現在位置の周辺のお店を直感的に探せます

---

### 8. 行きたい店を管理する

#### 行きたい店に追加する

1. 下部の **「行きたい店を追加」** ボタンをタップします
2. お店を検索して選択すると、「行きたい店」リストに登録されます

#### フォロワーを誘う

フィードのレビューカードや行きたい店にある **「誘う」** ボタンから、フォロワーに「一緒に行きたい」リクエストを送れます。相手が承諾すると、相手の行きたい店リストにも追加されます。

フィードの **「行きたい」タブ** で自分の行きたい店一覧を管理できます（ルート確認・削除も可能）。

---

## プロフィールを変更する

1. 下部メニューの **「プロフィール」** をタップします
2. ユーザー名・表示名を変更できます
3. 好きなジャンル（最大3つ）を変更できます
4. **「保存」** をタップして反映します

プロフィール画面の下部にある **「ログアウト」** ボタンからいつでもログアウトできます。

## よくある質問

### Q. フォローしていない人のレビューは見られますか？

見られません。GochiSpace は**相互フォロワー同士のみ**でレビューを共有する設計です。知らない人のレビューは一切表示されません。

### Q. フォロー申請を送ったあと、相手が承認するまでどうなりますか？

申請は「保留中」の状態になります。相手が承認するまでレビューは共有されません。

### Q. 自分のレビューを削除できますか？

できます。フィードでお店のカードをタップし、詳細表示にある自分のレビューのゴミ箱アイコンから削除できます。

### Q. 写真は何枚まで投稿できますか？

1回のレビュー投稿につき、最大**3枚**まで投稿できます。

### Q. ユーザー名はあとから変更できますか？

できます。プロフィール画面から変更可能です。ただし、ユーザー名は他のユーザーが検索に使うため、変更後は友人に新しいユーザー名を知らせてください。

### Q. フォローを解除したい場合は？

現在のバージョンではフォロー解除機能は実装予定です。
